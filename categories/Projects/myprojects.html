IEEE1588 Clock Synchronization

Undergraduate Thesis, Feb.-June 2018

  • Used C++ to implement logic of local state machines, which support message exchange, state decision and finally enable calibration of locol clock.
  • Enable RTP stream transmission without time drift using the software above.
